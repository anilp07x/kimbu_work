
# Kimbu_Work - Plataforma Flask minimalista para monitorar vagas de emprego em Angola.

## Funcionalidades:
- Web scraping com BeautifulSoup
- Página Flask responsivo (HTML + TailWind CSS)
- Base de dados SQLite
- Atualização automática (APScheduler)
- Notificações opcionais via email/Telegram

## Design:
- Minimalista, rápido, com foco em clareza e UX limpa.
- Paleta neutra (branco, cinza, azul suave), tipografia leve (Poppins/Inter).
